<h2>🛠️ Installation Steps:</h2>

<p>1. Clone repo</p>

```
https://github.com/morieskie/next-resume.git
```

<p>2. Install dependencies</p>

```
npm i
```

<p>3. Build project</p>

```
next build
```

<p>4. Code linting</p>

```
npm run lint
```

<p>5. Start project</p>

```
npm start
```
